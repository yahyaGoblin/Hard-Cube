# 🟥 HardCube

> My first game ever.

Built in Unity by following a Brackeys tutorial — with my own *slightly evil* twist.

---

## 🎮 About the Game

**HardCube** is a simple obstacle-based game where you control a cube and try to reach the end of each level without crashing.

* 🧱 6 levels
* ⚡ Increasing difficulty
* 🎯 Simple controls

---

## 💀 The Twist

> No checkpoints. No saves.

If you die at any point, you restart from **Level 1**.

You have to complete all 6 levels in **one run**.

---
